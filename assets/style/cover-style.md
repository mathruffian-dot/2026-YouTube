# YouTube 封面風格指南（三師爸頻道）

> 此檔是頻道封面的視覺基準。所有 `cover-image` Skill 呼叫的 prompt 都應依此風格撰寫，以維持頻道識別。

## 整體調性
**「Tech YouTuber × 教學頻道」風格**：高對比、深藍科技底、發光特效、強烈呼喚點擊感。
參考頻道氣口：勞利、科技狗、PAPAYA 電腦教室那種「教育型科技 YouTuber」的視覺語言。

---

## 五大固定元素

### 1. 背景（Background）
- **主色**：深藍 / 深海軍藍 / 接近黑
- **特效**：放射狀光線、霓虹漸層、電子粒子、細微網格、資料流線條
- **氣氛**：科技感、未來感，但不能太冷 — 要靠暖色（黃、橘）的標題拉回溫度
- **嚴禁**：純白底、純色平塗、自然風景、室內生活感

### 2. 主標題文字（Title Typography）
- **字體**：超粗黑體中文字（如思源黑體 Heavy / Noto Sans TC Black 等級）
- **主色**：**亮黃色**（首選）或**白色**，視構圖配色決定
- **描邊**：粗黑色描邊（4–6px），確保在任何背景下都清晰
- **特效**：可加輕微漸層（黃→橘）、輕微陰影或發光，**不要**3D 立體浮雕
- **位置**：左半邊或上方置中，佔畫面寬度 40–50%
- **行數**：2–3 行，**每行字數盡量一致**（視覺平衡）
- **字級**：標題字級要佔畫面高度的 1/3 以上 — 手機縮圖看得清楚

### 3. 副標題 / 集數標籤
- **集數標籤**（如「Claude 基本功 EP12」「Codex 基本功 EP02」）：
  - 位置：標題上方或下方，較小的字級
  - 配色：白色、淺藍色、或反白底色塊
  - 用於系列影片
- **副標**（描述影片內容）：
  - 通常較小，位於主標下方
  - 白色或淺色，加細邊
- **裝飾元素**：可加箭頭、星星、橫線、圓點等小圖標分隔

### 4. 人物（Persona）— 必用三師爸人物形象照
- **位置**：**畫面右側**（佔畫面寬 35–45%），偶爾正中
- **構圖**：半身或 3/4 身，臉部清晰可見
- **常見 pose**：
  - 雙手抱胸（沉穩）
  - 一手指向左方主題（教學）
  - 一手攤開呈現（介紹）
  - 雙手呈現產品（強調）
- **打光**：藍 / 青色邊緣補光（rim light），強化科技感與人物輪廓
- **嚴禁**：改變人物五官、髮型、外套顏色、眼鏡 → 永遠延續基準照

### 5. 主題視覺（Topic Visual）— 左側放置
- 與影片主題對應的圖示／插畫，例如：
  - AI Agent 主題 → 機器人、Agent 圖示、流程圖
  - Codex 主題 → 程式碼介面、終端機視窗
  - Claude 主題 → Claude logo、對話介面
  - 資料庫主題 → 倉庫、資料流、Database 圖示
- **發光效果**：主題視覺周圍要有發光暈，與背景區隔
- **配色**：與主標、背景呼應，常用青藍、橘黃、紫色發光

## 主題配色規則（重要）

**依影片內容主角的 AI 工具決定主色調：**

| 影片主角 | 主色調 | 主光線 | 主題視覺發光色 |
|---------|--------|--------|---------------|
| Claude（含 Claude Code、Claude Design）| **橘色** | 橘黃放射光 | 橘黃光暈 |
| GPT Codex（含 GPT-4、ChatGPT、Codex CLI）| **亮藍** | 電子青 / 亮藍放射光 | 青藍光暈 |
| 兩者並用 / 對比 / 整合 | **橘 + 藍**（雙色） | 一邊橘、一邊藍對比 | 兩種光暈左右分列 |
| 與 AI 工具無關（如純教學） | 預設亮黃 + 電子青 | 電子青 | 青藍 |

決策範例：
- 「Claude 基本功 EP01」→ 橘
- 「Codex 基本功 EP02」→ 藍
- 「Claude × Codex 整合教學」→ 橘 + 藍
- 「用 Claude Agent 寫課程計畫」→ 橘

## 色票（Color Palette）

### 背景與通用
| 用途 | 色名 | Hex |
|------|------|-----|
| 背景主色 | 深海軍藍 | `#0A1628` ~ `#0F2647` |
| 主標題 | 亮黃 | `#FFD700` ~ `#FFC107` |
| 主標題（次選）| 純白 | `#FFFFFF` |
| 標題描邊 | 純黑 | `#000000` |
| 副標 / 細節 | 淺藍 / 白 | `#A5D8FF` / `#FFFFFF` |

### Claude 主題（橘）
| 用途 | 色名 | Hex |
|------|------|-----|
| 放射光 / 光暈 | Claude 橘 | `#FF8C42` |
| 強調光 | 暖橘金 | `#FFA500` |
| 點綴 | 深橘紅 | `#FF6B35` |

### Codex 主題（藍）
| 用途 | 色名 | Hex |
|------|------|-----|
| 放射光 / 光暈 | 電子青 | `#00D4FF` |
| 強調光 | 亮藍 | `#0099FF` |
| 點綴 | 紫光 | `#9D4EDD` |

### 雙色主題（Claude × Codex）
左半邊用 Codex 藍系、右半邊用 Claude 橘系，中間自然漸層融合。
（人物身上的 rim light 可一邊藍、一邊橘以呼應主題。）


例外：人物可微微越過中線，但臉部中心應在右半邊。

## Prompt 撰寫範本

填空版本：
```
YouTube 封面 16:9，三師爸頻道科技教學風格。

背景：深海軍藍 (#0A1628) 漸層底，左下與右上有「{THEME_LIGHT_COLOR}」放射光，
細微網格與粒子特效，整體科技未來感。
※「{THEME_LIGHT_COLOR}」依影片主角決定（見上方主題配色規則）：
   - Claude → 橘 (#FF8C42)
   - Codex → 電子青 (#00D4FF)
   - 兩者並用 → 左藍右橘對比

人物：畫面右側放置這位男性教師（戴眼鏡、黑色連帽外套），姿勢為「{POSE}」，
青藍色 rim light 打在邊緣，臉部表情「{EXPRESSION}」。
※ 人物樣貌、五官、髮型、外套、眼鏡全部延續輸入的人物基準照，不要更動。

主視覺（左半邊）：{TOPIC_VISUAL}，周圍有發光暈與背景區隔。

文字：
- 集數標籤（左上）：「{SERIES_LABEL}」白色細字
- 主標題（左中）：「{MAIN_TITLE}」亮黃色 (#FFD700) 超粗黑體中文字，
  粗黑色描邊（4–6px），佔畫面寬度 40–50%，2–3 行排列。
- 副標題（主標下方）：「{SUBTITLE}」白色細字，較小字級。

整體風格：高對比、強烈呼喚點擊感、適合教師受眾。
不要：純白底、自然風景、寫實生活感、3D 浮雕字。
```

填空變數：
- `{THEME_LIGHT_COLOR}`：依影片主角的 AI 工具決定（橘 / 青藍 / 雙色）
- `{POSE}`：例「雙手抱胸沉穩」/「一手指向左側主題」/「雙手呈現筆電」
- `{EXPRESSION}`：例「自信微笑」/「專注解說」/「友善看向鏡頭」
- `{TOPIC_VISUAL}`：例「Claude AI 介面正在自動填寫表格」
- `{SERIES_LABEL}`：例「Claude 基本功 EP01」/ 留空
- `{MAIN_TITLE}`：YouTube 影片標題（精簡版）
- `{SUBTITLE}`：副標說明（可選）

## 標準呼叫指令

```bash
python skills/cover-image/draw.py "<填好的 prompt>" \
  --edit "assets/persona/三師爸人物形象照.png" \
  --size 1536x1024 --quality low \
  --name cover --outdir "output/<標題>/"
```

quality 一律 **low**（夠用、便宜、快），除非要實體印刷才升 high。
