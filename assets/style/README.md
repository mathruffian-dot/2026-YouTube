# 封面風格資料夾

放兩個檔案，定義**你的頻道**封面視覺識別。

## 1. `cover-style.md`（必填）
完整的封面風格指南：色票、構圖、字體、主題配色規則、prompt 範本。

**這份要完整改成你自己頻道的版本。** 預設裡面的「Claude=橘 / Codex=藍」是 mathruffian 頻道的選擇，未必適合你。

建議你保留這份的**結構**，但把以下內容換成你的：
- 整體調性描述
- 背景配色（深 vs 亮）
- 字體選擇（粗黑 vs 圓潤）
- 人物位置（左 vs 右 vs 置中）
- 主視覺類型（icon / 截圖 / 插畫）
- 主題配色規則（你的頻道有沒有「主角=配色」這種規則？）

## 2. `reference-thumbnails.png`（強烈建議）
你頻道現有的 9–12 張封面截圖，拼成一張圖。

**為什麼有這個？** 生封面前 AI Agent 會用 Read 工具看這張，**內化你的視覺風格**，再寫出貼近的 prompt。比純文字描述更精準。

### 怎麼做？
1. 打開你的 YouTube 頻道首頁
2. 截一張 12 格的封面截圖
3. 存成 `reference-thumbnails.png` 放這裡

### 沒有歷史封面怎麼辦？
- 新頻道：跳過這檔，依賴 `cover-style.md` 的文字描述就夠
- 想參考別人的風格：截別人頻道的封面也行，但建議寫進 `cover-style.md` 註明「希望走 XX 風格」
